# Security Policy

## Supported Versions

We maintain the latest `main` branch.

## Reporting a Vulnerability

If you find a security issue in web3-security-scout, please email security@gboyee.dev.

We aim to respond within 48 hours and provide a remediation plan within 7 days.

## Recommendations for Users

- Keep API keys (Etherscan, Alchemy) secret; use `.env` and never commit them.
- Run scanner in a sandboxed environment if testing untrusted contracts.
- Update dependencies regularly.
