Case Study: Autonomous GitHub Agent

Problem: Developers waste hours on manual PR reviews, leading to bottlenecks and inconsistent quality.

Solution: Built an AI agent that fetches PR diffs via GitHub API, analyzes code using a local LLM (Ollama), and posts structured review comments automatically. Includes secure webhook validation and rate limiting.

Results: Review time reduced by ~80%, with consistent detection of bugs and security issues. The agent runs autonomously and integrates seamlessly into existing workflows.

Tech: Python, Ollama (qwen2.5-coder:7b), Nginx, GitHub API, HMAC security.
