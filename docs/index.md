# Web3 Security Scout

Automated bug bounty scanner for Web3 protocols.

## Features
- Continuous monitoring of Immunefi, HackerOne
- Smart contract analysis (Solidity)
- PoC generation for eligible bounties
- Integration with OpenClaw hive

## Quickstart
```bash
pip install -r requirements.txt
python -m web3_security_scout.scan
```

See configuration guide to set API keys.
