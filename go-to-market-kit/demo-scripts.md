## 🎬 Demo 1 — PR Reviewer (Money Maker)

**Length:** 60 seconds

**Script:**

> “This is an AI agent I built that reviews GitHub pull requests automatically.”

**Scene sequence:**

1. Open a GitHub PR with code changes (use a public repo or test repo)
2. Terminal: run `xander-pr-reviewer https://github.com/owner/repo/pull/123`
3. Fast‑forward (time‑lapse) showing terminal output: “Fetching PR… generating review…”
4. Switch to GitHub → PR → Comments tab
5. Show AI comment: structured review with Summary, Bugs, Security, Suggestions, Verdict
6. Point out: “It found a potential security issue and suggested a fix” (highlight)
7. End screen with text overlay:
   “Autonomous PR review in ~2 hours. No human in the loop.”
   CTA: “Get early access → link in bio”

- Highlight accuracy

## 🎬 Demo 3 — Full Automation Loop

**Length:** 90 seconds

**Script:**

> “This is what AI‑powered DevOps looks like: from bug report to merge, zero human touch.”

**Scene sequence:**

1. GitHub issue created: “Memory leak in process handler”
2. Terminal: agent detects new issue, clones repo, analyzes code
3. Agent creates fix branch, commits changes, opens PR
4. Xander PR Reviewer automatically reviews that PR (Demo 1)
5. If review is APPROVE, agent merges automatically
6. Show issue closed automatically
7. End screen:
   “Bug → Fix → Review → Merge — automated.”
   CTA: “Hire me to build your autonomous DevOps → link in bio”

**Voiceover:**
- “Issue arrives → agent wakes up”
- “It codes the fix, opens PR”
- “Another agent reviews”
- “If good, it merges — closed loop”
